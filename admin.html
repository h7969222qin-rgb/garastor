First-Time Setup — Generate token.enc

Enter a GitHub Personal Access Token (read/write to contents on this repo) and your chosen dashboard password. This encrypts the token with your password and downloads token.enc.

How to create the GitHub Token

1. GitHub → Settings → Developer settings → Personal access tokens → Fine-grained → Generate.
2. Resource owner: your account. Repository: Only select → garastro.
3. Permissions → Contents: Read and write. Metadata: Read-only.
4. Generate, copy the token (github_pat_...), paste above.
Or use a classic token with the repo scope.

First-Time Setup — Generate token.enc

Run this once to enable login. Enter a GitHub Personal Access Token (with read/write to contents on this repo) and your chosen dashboard password. It encrypts the token with your password and downloads token.enc. Commit that file into the repo (root + dist).

How to create the GitHub Token

1. Go to GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate.
2. Resource owner: your account. Repository access: Only select repositories → garastro.
3. Permissions → Repository permissions → Contents: Read and write. Metadata: Read-only.
4. Generate, copy the token (starts with github_pat_), paste above.
Or use a classic token with the repo scope.
